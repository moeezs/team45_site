Client Report

Project Name: Bounce

Client Name: Krishh Lala

Target Users:

The primary users of this web app are basketball enthusiasts who enjoy playing strategy-based sports simulations. These users may range from casual fans looking for a fun way to engage with the sport, to more dedicated basketball analysts who want to test their strategic decision-making. The app will likely attract desktop users due to its strategic and text-based nature, but ensuring mobile compatibility would be beneficial for accessibility.

Static Information:

The app should provide basic information including:

  • FAQ/help section explaining the game mechanics
  • Leaderboards showing top-performing users
  • A database of real NBA players with their respective stats

Design Elements:

The design of the web app should be clean, intuitive, and focused on text-based interactions. Since the simulation relies on user decisions rather than visual animations, a structured and easy-to-navigate interface is essential. Key design elements should include clearly labeled options for each play action, a visually appealing but minimalistic UI to keep users engaged, avoid overly flashy visuals or unnecessary animations, and must focus on readability, ensuring that stats and play choices are easy to access.

Stored Data:

The web app will need to manage and present various types of data, including:

  • Player Database: Real-life player stats, such as shooting accuracy, defensive ability, and stamina, which influence gameplay.
  • Team Management System: Users can create and manage teams based on real players.
  • Game Simulation Data: The app needs to track in-game progress, scoring, and player actions based on probabilities derived from real statistics.
  • User Currency System: A virtual currency system where players earn rewards for winning games and can use these rewards for team upgrades or other enhancements.

Interactive Elements:

While the game is primarily text-based, several interactive elements can help enhance user engagement:

  • Choice-Based Gameplay: Users select actions (e.g., shooting a three-pointer, driving for a layup, playing tight defense) that influence game outcomes based on real stats.
  • Dynamic Probability-Based Outcomes: Player stats dictate success rates, ensuring a realistic and balanced simulation experience.
  • Currency & Rewards System: Users earn currency for victories, which can be used to acquire better players, improve team stats, or unlock special features.
  • Customizable Teams: Users draft real players, forming unique teams based on real-world abilities and strategic preferences.
